# v3-vite-ts-blog
框架：vuejs ^3.4.21
css解决方案：sass ^1.72.0
状态管理方案：pinia ^2.0.11
路由管理方案：vue-router ^4.3.0
代码格式化：prettier ^3.2.5
代码检查：eslint ^8.57.0
打包工具/脚手架：vite ^5.1.6


## 安装依赖
```javascript
git clone https://github.com/Gary-Leee/v3-vite-ts-blog.git  

npm i or pnpm i（推荐）

npm start or pnpm start
```

## 项目介绍

因为是纯前端项目，所以安装完依赖后可以直接部署在云服务器上当personal profile
