# v3-vite-ts-blog
vue3+vite+ts的blog

## 安装依赖
```javascript
git clone https://github.com/Gary-Leee/v3-vite-ts-blog.git  

npm i 或者 pnpm i（推荐）

npm start
```

## 项目介绍

因为是纯前端项目，所以安装完依赖后可以直接部署在云服务器上当personal profile

在线体验网址：[传送地址](http://garyleee.cool/ "Gary leee")
